# SimpleOTP
Simple One Time Pad Progressive Web App
--------------------
You can load your own random numbers.
Do not use the same set of random numbers for more than one message. Use a new set of random numbers and a new unique key for each new message.
Messages could use a word list. 
https://www.eff.org/files/2016/07/18/eff_large_wordlist.txt
https://www.dcode.fr/pgp-word-list#f1
